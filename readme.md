## Requirements

- obs (`sudo apt install obs-studio`)
- [obs-ndi](https://github.com/obs-ndi/obs-ndi/releases)
  - Install `libndi` and `obs-ndi`
- `obs --minimize-to-tray --websocket_port 3002 --websocket_password testtest --websocket_ipv4_only`
